# Installation



{% hint style="danger" %}
**coronabot.js** only works on **discord.js@12**
{% endhint %}

```text
npm install --save coronabot.js
```

{% hint style="info" %}
Installing latest version
{% endhint %}

You have a little buggy version, **but** works on **discord.js@11**
{% endhint %}

```text
npm install --save coronabot.js@1.0.0
```
